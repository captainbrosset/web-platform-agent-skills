# Sets

Web platform features about Sets

## Sets features

### Set (initial support)

Set objects store unique values of any type.

**Spec:**
- <https://tc39.es/ecma262/multipage/keyed-collections.html#sec-set-objects>

**Browser support:**
- Chrome: 38
- Chrome Android: 38
- Edge: 12
- Firefox: 13
- Firefox for Android: 14
- Safari: 8
- Safari on iOS: 8

**MDN:**
- [Set](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Set)

### Set methods

The `difference()`, `intersection()`, `isDisjointFrom()`, `isSubsetOf()`, `isSupersetOf()`, `symmetricDifference()`, and `union()` methods of the JavaScript `Set` object performs operations between two sets.

**Spec:**
- <https://tc39.es/ecma262/multipage/keyed-collections.html#sec-properties-of-the-set-prototype-object>

**Browser support:**
- Chrome: 122
- Chrome Android: 122
- Edge: 122
- Firefox: 127
- Firefox for Android: 127
- Safari: 17
- Safari on iOS: 17

**MDN:**
- [Set.prototype.difference()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Set/difference)
- [Set.prototype.intersection()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Set/intersection)
- [Set.prototype.isDisjointFrom()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Set/isDisjointFrom)
- [Set.prototype.isSubsetOf()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Set/isSubsetOf)
