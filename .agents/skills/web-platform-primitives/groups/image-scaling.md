# Image scaling

The `image-rendering` CSS property sets how images are scaled.

## Image scaling features

### crisp-edges

The `image-rendering: crisp-edges` CSS declaration scales images to preserve lines without blurring.

**Spec:**
- <https://drafts.csswg.org/css-images-3/#the-image-rendering>

**Browser support:**
- Chrome: —
- Chrome Android: —
- Edge: —
- Firefox: 65
- Firefox for Android: 65
- Safari: 7
- Safari on iOS: 7

**MDN:**
- [image-rendering](https://developer.mozilla.org/docs/Web/CSS/Reference/Properties/image-rendering)

### image-rendering

The `image-rendering` CSS property sets how images are scaled, retaining smoothness for photos, or hard edges for pixel art and QR codes.

**Spec:**
- <https://drafts.csswg.org/css-images-3/#the-image-rendering>

**Browser support:**
- Chrome: 41
- Chrome Android: 41
- Edge: 79
- Firefox: 93
- Firefox for Android: 93
- Safari: 10
- Safari on iOS: 10

**MDN:**
- [image-rendering](https://developer.mozilla.org/docs/Web/CSS/Reference/Properties/image-rendering)

### image-rendering: smooth

The `image-rendering: smooth` CSS declaration scales images while retaining a gradual color blending that is useful for photos.

**Spec:**
- <https://drafts.csswg.org/css-images-3/#the-image-rendering>

**Browser support:**

- Chrome Android: —
- Edge: —

- Firefox for Android: 93
- Safari: —
- Safari on iOS: —
