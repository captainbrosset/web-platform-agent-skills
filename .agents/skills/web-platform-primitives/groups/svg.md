# SVG

SVG (Scalable Vector Graphics) is a language for describing two-dimensional graphics. spec: https://svgwg.org/svg2-draft/

## SVG features

### context-fill and context-stroke

The `context-fill` and `context-stroke` CSS values are used with `fill` and `stroke` in SVG `<marker>` or `<use>` elements to match a color from where they appear.

**Spec:**
- <https://svgwg.org/svg2-draft/painting.html#SpecifyingPaint>

**Browser support:**
- Chrome: 124
- Chrome Android: 124
- Edge: 124
- Firefox: 111
- Firefox for Android: 111
- Safari: —
- Safari on iOS: —

**MDN:**
- [fill](https://developer.mozilla.org/docs/Web/SVG/Reference/Attribute/fill)
- [stroke](https://developer.mozilla.org/docs/Web/SVG/Reference/Attribute/stroke)

### Opacity (SVG)

The `fill-opacity`, and `stroke-opacity` SVG attributes and CSS properties control the transparency of a stroke or fill of an SVG element.

**Spec:**
- <https://svgwg.org/svg2-draft/render.html#ObjectAndGroupOpacityProperties>

**Browser support:**
- Chrome: ≤80
- Chrome Android: ≤80
- Edge: ≤80
- Firefox: 1.5
- Firefox for Android: 4
- Safari: ≤13.1
- Safari on iOS: ≤13.4

**MDN:**
- [fill-opacity](https://developer.mozilla.org/docs/Web/SVG/Reference/Attribute/fill-opacity)
- [stroke-opacity](https://developer.mozilla.org/docs/Web/SVG/Reference/Attribute/stroke-opacity)

### SMIL SVG animations

The `<animate>`, `<animateMotion>`, and `<animateTransform>` SVG elements declaratively animate SVG elements. Also known as SMIL.

**Spec:**
- <https://svgwg.org/specs/animations/>

**Browser support:**
- Chrome: 19
- Chrome Android: 25
- Edge: 79
- Firefox: 4
- Firefox for Android: 4
- Safari: 6
- Safari on iOS: 6

**MDN:**
- [SVG animation with SMIL](https://developer.mozilla.org/docs/Web/SVG/Guides/SVG_animation_with_SMIL)

### SVG

The SVG image format, represented by the `<svg>` element, creates two-dimensional vector graphics with declarative or scripted interaction and animation.

**Spec:**
- <https://svgwg.org/svg2-draft/>

**Browser support:**
- Chrome: 1
- Chrome Android: 18
- Edge: 79
- Firefox: 3
- Firefox for Android: 4
- Safari: 3
- Safari on iOS: 2

**MDN:**
- [SVG: Scalable Vector Graphics](https://developer.mozilla.org/docs/Web/SVG)
- [`<svg>`](https://developer.mozilla.org/docs/Web/SVG/Reference/Element/svg)
- [SVG element reference](https://developer.mozilla.org/docs/Web/SVG/Reference/Element)

### SVG filters

The `<filter>` SVG element applies custom effects such as color manipulation, blurring, or morphing to SVG elements.

**Spec:**
- <https://drafts.csswg.org/filter-effects-1/>

**Browser support:**
- Chrome: 5
- Chrome Android: 18
- Edge: 12
- Firefox: 3
- Firefox for Android: 4
- Safari: 6
- Safari on iOS: 6

**MDN:**
- [`<filter>`](https://developer.mozilla.org/docs/Web/SVG/Reference/Element/filter)
