# Maps

Web platform features about Maps

## Maps features

### Array grouping

The `Object.groupBy()` and `Map.groupBy()` static methods group values of arrays and iterables based on a function that returns a key for each value.

**Spec:**
- <https://tc39.es/ecma262/multipage/abstract-operations.html#sec-groupby>

**Baseline Newly Available since 2024-03-05**

**MDN:**
- [Map.groupBy()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Map/groupBy)
- [Object.groupBy()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object/groupBy)

### Map getOrInsert()

The `getOrInsert()` and `getOrInsertComputed()` methods of `Map` objects get a value, setting and getting a default value if needed.

**Spec:**
- <https://tc39.es/ecma262/multipage/keyed-collections.html#sec-map.prototype.getorinsert>

**Baseline Newly Available since 2026-02-14**

**MDN:**
- [Map.prototype.getOrInsert()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Map/getOrInsert)
- [Map.prototype.getOrInsertComputed()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Map/getOrInsertComputed)
- [WeakMap.prototype.getOrInsert()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WeakMap/getOrInsert)
- [WeakMap.prototype.getOrInsertComputed()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WeakMap/getOrInsertComputed)
