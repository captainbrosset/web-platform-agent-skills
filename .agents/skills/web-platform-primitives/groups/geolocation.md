# Geolocation

Web platform features about Geolocation

## Geolocation features

### `<geolocation>`

The `<geolocation>` HTML element represents a button that, upon activation, prompts the user to choose whether to grant the page access to geolocation data.

**Spec:**
- <https://wicg.github.io/PEPC/geolocation-element.html>

### Geolocation API

The `navigator.geolocation` API requests the user's latitude, longitude, heading, and speed.

**Spec:**
- <https://w3c.github.io/geolocation/>

**Browser support:**
- Chrome: 5
- Chrome Android: 18
- Edge: 12
- Firefox: 3.5
- Firefox for Android: 4
- Safari: 5
- Safari on iOS: ≤3

**MDN:**
- [Geolocation API](https://developer.mozilla.org/docs/Web/API/Geolocation_API)
