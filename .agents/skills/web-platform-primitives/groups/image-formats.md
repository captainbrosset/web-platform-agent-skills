# Image formats

The various image formats supported by browsers.

## Image formats features

### AVIF

AVIF (AV1 Image File Format) is an image format based on the AV1 video format.

**Spec:**
- <https://aomediacodec.github.io/av1-avif/>

**Baseline Newly Available since 2024-01-25**

**MDN:**
- [Image file type and format guide](https://developer.mozilla.org/docs/Web/Media/Guides/Formats/Image_types)

### JPEG XL

The JPEG XL image format is a raster graphics file format that supports animation, alpha transparency, and lossy as well as lossless compression.

**Spec:**
- <https://www.iso.org/standard/85253.html>
