# Flexbox

https://drafts.csswg.org/css-flexbox-1/

## Flexbox features

### Flexbox

Flexbox is a one-dimensional layout system, which places content either horizontally or vertically, with optional wrapping.

**Spec:**
- <https://drafts.csswg.org/css-flexbox-1/>

**Browser support:**
- Chrome: 29
- Chrome Android: 29
- Edge: 12
- Firefox: 20
- Firefox for Android: 20
- Safari: 9
- Safari on iOS: 9

**MDN:**
- [Flexbox](https://developer.mozilla.org/docs/Learn_web_development/Core/CSS_layout/Flexbox)

### Flexbox gap

The `gap` CSS property in a flexbox layout sets the size of the space between items.

**Spec:**
- <https://drafts.csswg.org/css-align-3/#gaps>

**Browser support:**
- Chrome: 84
- Chrome Android: 84
- Edge: 84
- Firefox: 63
- Firefox for Android: 63
- Safari: 14.1
- Safari on iOS: 14.5

**MDN:**
- [gap](https://developer.mozilla.org/docs/Web/CSS/Reference/Properties/gap)

### Gap decorations

The `column-rule` and `row-rule` CSS properties display decorative lines between columns and rows of a flex, grid, or multi-column layout. The `rule-break`, `rule-outset`, and `rule-paint-order` properties control the appearance of these lines.

**Spec:**
- <https://drafts.csswg.org/css-gaps-1/>
